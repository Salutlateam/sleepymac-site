Politique de confidentialité

Dernière mise à jour : 27 février 2026

1. Éditeur

SleepyMac est édité par Alexandre Tapiero, développeur indépendant basé en France.

Contact : alexandre.tapiero@gmail.com

2. Données collectées

SleepyMac collecte le strict minimum de données nécessaires à son fonctionnement :

  • Clé de licence — stockée localement sur votre Mac dans un fichier chiffré (~/.sleepymac). Transmise à Lemon Squeezy uniquement pour la validation.
  • Identifiant d'instance — un identifiant unique généré lors de l'activation, utilisé pour le comptage des appareils.

SleepyMac ne collecte pas :

  • Vos habitudes d'utilisation
  • Vos données personnelles (nom, email, adresse)
  • Vos fichiers ou données système
  • Aucune donnée de tracking ou analytics dans l'app

3. Communication réseau

SleepyMac communique uniquement avec api.lemonsqueezy.com pour :

  • Activer votre licence (une fois à l'achat)
  • Valider votre licence (une fois par jour)
  • Désactiver votre licence (si vous changez d'appareil)

Ces communications sont chiffrées via HTTPS avec épinglage de certificat SSL (certificate pinning).

4. Paiement

Les paiements sont gérés par Lemon Squeezy (Lemon Squeezy LLC). SleepyMac n'a jamais accès à vos données bancaires. Consultez la politique de confidentialité de Lemon Squeezy.

5. Stockage des données

Toutes les données de SleepyMac sont stockées localement sur votre Mac dans :

  • ~/Library/Application Support/SleepyMac/.license

Aucune donnée n'est stockée sur nos serveurs.

6. Site web

Le site sleepymac.app est hébergé sur Vercel. Il n'utilise pas de cookies. Si le service d'analytics Plausible est activé, il est sans cookie et conforme au RGPD.

7. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données, vous disposez des droits suivants :

  • Droit d'accès — demander quelles données nous détenons sur vous
  • Droit de suppression — demander la suppression de vos données
  • Droit de portabilité — recevoir vos données dans un format lisible

Pour exercer ces droits, contactez : alexandre.tapiero@gmail.com

8. Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut de cette page.
